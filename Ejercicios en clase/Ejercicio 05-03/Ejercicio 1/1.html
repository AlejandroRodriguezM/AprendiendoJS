Modelos de eventos en JavaScript

JavaScript ofrece tres modelos diferentes para manejar eventos:

  1. Modelo de eventos de captura: En este modelo, el evento se propaga desde el elemento más externo hasta el más interno, es decir, desde el padre hasta el hijo. Puede ser manipulado en cualquier momento durante la propagación.
  2. Modelo de eventos de bubbling: Este modelo funciona al revés del modelo de captura. El evento comienza en el elemento más interno y se propaga hacia afuera. Este modelo es más utilizado en la práctica.
  3. Modelo de eventos de ambos: Este modelo combina los dos modelos anteriores y permite manipular el evento tanto durante la captura como durante la burbuja.
